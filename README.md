#  AI-Powered Healthcare Diagnosis Assistant

##  Project Overview

AI-Powered Healthcare Diagnosis Assistant is a Machine Learning based web application that predicts possible diseases based on the patient's age, gender, and selected symptoms.

The project is developed using Python, Flask, Scikit-learn, HTML, and CSS. It provides disease prediction along with confidence score, disease description, recommended medicine, and basic precautions.

---

##  Features

- Disease Prediction using Machine Learning
- Patient Age and Gender Input
- Symptom Selection
- Confidence Score
- Disease Description
- Recommended Medicine
- Precautions
- User-Friendly Interface
- Responsive Design

##  Installation

1. Clone or download the project.

2. Install required libraries:

```bash
pip install flask pandas numpy scikit-learn joblib
```

3. Run the application:

```bash
python app.py
```

4. Open your browser and visit:

```
http://127.0.0.1:5000
```

---

##  Workflow

1. Enter Patient Age.
2. Select Gender.
3. Select Symptoms.
4. Click **Predict Disease**.
5. View:
   - Predicted Disease
   - Confidence Score
   - Disease Description
   - Recommended Medicine
   - Precautions
